## Installation

1. Clone repo
2. cd server
3. run `npm install`
4. go to config folder and make a .env file
DB_STRING = 
PORT = 
CLOUDINARY_API_KEY = 
CLOUDINARY_API_SECRET = 
CLOUDINARY_NAME = 
5. add all those variables to the .env file

## Usage
1, run `npx tsc` 
2. run `node server.js`
